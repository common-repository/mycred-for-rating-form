== Description ==

Choose or create a [Rating Form](https://wpexperts.io/link/rating-form/), choice of Stars, Hearts, Smileys, Thumbs Up & Down and many more or upload your own shapes. Lots of features and still expanding. This plugin allows you to reward users with points for submitting ratings.

= myCred for Rating Form Features =
Here are the most notable features of myCred for Rating Form.

* Award points for ratings.
* Badge support.
* Translation ready.

= Plugin Requirements =

* [myCred 1.8+](https://wordpress.org/plugins/mycred/)
* [Rating Form](https://wpexperts.io/link/rating-form/)
* WordPress 5.0+
* PHP 5.3+

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for Rating Form
3. Find myCred for Rating Form and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
